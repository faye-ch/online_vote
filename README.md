# 1.技术说明

1.前端：vue+thymeleaf+bootstrap+echarts

2.工程管理:maven

3.框架 :SpringBoot 

4.简化代码 : lombok插件

5.加密：MD5 

6.Json工具：fastjson



# 2.功能介绍

1.用户免登录投票，根据用户IP防止刷票，查看投票结果

2.会员注册登录，发表投票、修改投票、管理投票

3.数据分析，会员查看自己的投票结果，根据用户IP分析用户来源
